# Expense-Tracker-


A clean and minimal expense tracking web application built using React.  
The application allows users to add and delete expenses, calculate total spending, and persist data using the browser's LocalStorage.

---

## Features

- Add new expenses
- Delete existing expenses
- Automatic total calculation
- Persistent storage using LocalStorage
- Responsive user interface
- Built with modern React practices

---

## Tech Stack

- React
- JavaScript (ES6+)
- Vite
- Styled Components
- LocalStorage API

---

## Installation and Setup

Clone the repository:

git clone  https://github.com/kamalesh8119/expense-tracker.git

Navigate to the project folder:

cd expense-tracker

Install dependencies:

npm install

Run the development server:

npm run dev
